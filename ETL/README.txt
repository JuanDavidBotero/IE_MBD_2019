Welcome to the offshore_data.xlsx Windfarm Performance database ETL process.


==============================================================================================================

To initiate the ETL process:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1- Please run the Offshore_Windfarm.mwb file to ensure that the database scheme has been created. The scheme uses the database name mydb, so please ensure that that name is not being used prior to launching this file.

2- Run the JB_Full_ETL.kjb file to start the ETL process.

3- Enjoy your populated database

===============================================================================================================

Technical requirements to execute a successful data upload: 
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- ETL tool: Pentaho Data Integration 

- Assign a 4GB memory capacity to Pentaho, so there are no Overhead capacity errors. 


===============================================================================================================

Metrics of the data upload: 

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- Total time to upload the data to the Data Warehouse: 8 to 10 minutes depending on the Operating System and hardware.
